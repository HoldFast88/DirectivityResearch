DirectivityResearch
===================


This project devoted to directionsl microphones research. It can work with 3 types of microphones:
1. Parabolic;
2. Organ;
3. Linear group.
For these types it can build directivity diagrams, directional frequency references and compare it, determine maximum working frequency.


This project contains libraries with specified dependences:
- ZedGraph - .NET Framework v.2.0+
- MathNet - .NET Framework v.4.0+
- alglibnet2 - C# 2.0
